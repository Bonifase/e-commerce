## Installation

OS X & Linux:

```sh
pip install django --save
```

## Development setup

> Follow the steps below.

```sh
$ git clone https://github.com/Bonifase/e-commerce.git.
cd into the folder e-commerce
pip install django --save
python manage.py collectstatic
python manage.py runserver
```
